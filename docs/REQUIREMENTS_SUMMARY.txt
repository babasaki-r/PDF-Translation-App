1. システム要件
================================================================================

【ハードウェア】
- Mac: Apple Silicon (M1以降) 必須
- メモリ: 最小16GB、推奨32GB以上
- ストレージ: 最小20GB空き

【OS】
- macOS 14.0 (Sonoma) 以上
- Apple翻訳（簡易翻訳）: macOS 15.0 (Sequoia) 以上

================================================================================
2. 必要なソフトウェア
================================================================================

【必須】
- Python 3.9以上 (推奨: 3.11)
- Node.js 18以上 (推奨: 20 LTS)
- Homebrew (パッケージ管理)

【オプション（翻訳エンジン別）】
- Ollama (バランス翻訳に必要)
- Xcode Command Line Tools (Apple翻訳に必要)

6. クイックセットアップコマンド
================================================================================

# 1. Homebrewインストール（未インストールの場合）
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 2. 必要なソフトウェアをインストール
brew install python@3.11 node@20 ollama

# 3. Ollamaモデルをダウンロード
ollama pull qwen3:4b

# 4. プロジェクトをセットアップ
cd PDF_Translation_App
./setup.sh

# 5. 起動
ollama serve &  # 別ターミナル推奨
./start.sh

8. 翻訳エンジン別の追加要件
================================================================================

【バランス（Ollama）】
- Ollamaのインストール必須
- ollama serveの起動必須
- qwen3:4bモデルのダウンロード必須

【日本語重視（Swallow）】
- 初回起動時にHugging Faceからモデル自動ダウンロード（約14GB）
- メモリ8GB以上の空きが必要

【簡易翻訳（Apple翻訳）】
- macOS 15.0 (Sequoia) 以上
- Xcode Command Line Tools
- 初回使用時に言語パックのダウンロードが必要な場合あり
