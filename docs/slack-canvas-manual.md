# PDF Translation App マニュアル

## 概要

英語の技術文書PDFを日本語に翻訳するアプリケーションです。
**ローカルAI**で動作するため、機密文書も安全に翻訳できます。

---

## 主な機能

| 機能 | 説明 |
|------|------|
| PDF翻訳 | 英語PDFを日本語に自動翻訳 |
| 3つの翻訳エンジン | Ollama / Transformers / Apple翻訳 |
| 用語集 | 専門用語を登録して翻訳の一貫性を保持 |
| 質問機能 | PDFの内容についてAIに質問 |
| 再翻訳 | エンジンやモデルを変更して再翻訳 |

---

## 起動方法

### 1. Ollamaを起動（推奨エンジン使用時）
```
ollama serve
```

### 2. アプリを起動
```
cd ~/PDF_Translation_App
./start.sh
```
ブラウザが自動で開きます → http://localhost:5173

### 3. 終了
```
./stop.sh
```

## 翻訳エンジンの選択

ヘッダー右側のドロップダウンから選択できます。

| エンジン | 特徴 | 推奨用途 |
|----------|------|----------|
| **Ollama** | 高速・省メモリ・高品質 | 通常の翻訳（推奨） |
| **Transformers** | 高品質だが重い | オフライン環境 |
| **Apple翻訳** | 軽量・高速 | 簡易翻訳 |

### Ollamaモデル一覧

| モデル | 特徴 |
|--------|------|
| Qwen3 4B | 高速・軽量（推奨） |
| Qwen2.5 7B | バランス型 |
| Llama 3.1 8B | 汎用モデル |
| Qwen3 14B | 高品質 |

## 基本的な使い方

### Step 1: PDFをアップロード
- 「ファイルを選択」ボタンをクリック
- またはドラッグ＆ドロップ

### Step 2: 翻訳エンジン・モデルを選択
- ヘッダーのドロップダウンで選択

### Step 3: 翻訳を開始
- 「翻訳を開始」ボタンをクリック
- 進捗バーで状況を確認

### Step 4: 結果を確認
- 左側: PDF原文
- 右側上部: 英語テキスト
- 右側下部: 日本語翻訳

---

## 再翻訳機能

翻訳完了後、別のエンジンやモデルで再翻訳できます。

1. ヘッダーでエンジン/モデルを変更
2. **「🔄 再翻訳」ボタン**をクリック
3. 確認ダイアログで「OK」

**活用シーン:**
- 翻訳品質に満足できない → 別モデルで再翻訳
- 用語集を更新した後 → 再翻訳で反映
- 高速モデルで確認後 → 高品質モデルで本番翻訳

---

## 用語集機能

画面右下の「用語集」ボタンから開きます。

### 用語の追加
1. 英語欄に英語の用語を入力
2. 日本語欄に対応する訳語を入力
3. 「追加」ボタンをクリック

### ポイント
- 用語集は自動保存されます
- 更新後は「🔄 再翻訳」で反映

---

## 質問機能

画面右下の「?」ボタンから開きます。

### 使い方
1. 質問を入力
2. 「質問する」ボタンをクリック
3. AIが回答を生成（10〜30秒）

### 質問例
- 「この文書の概要を教えてください」
- 「〇〇とは何ですか？」
- 「主な仕様を要約してください」

## トラブルシューティング

| 症状 | 対処法 |
|------|--------|
| Ollamaが利用できない | `ollama serve` でサーバーを起動 |
| 翻訳が遅い | 軽量モデル（Qwen3 4B）またはApple翻訳を使用 |
| 中国語が混じる | 「🔄 再翻訳」ボタンで翻訳し直す |
| メモリ不足 | Ollamaまたは軽量モデルに変更 |
| PDFが読み込めない | パスワード保護やファイル破損を確認 |
